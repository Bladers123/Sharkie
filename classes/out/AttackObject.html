Class: AttackObject

AttackObject(positionX, positionY, otherDirection, type, range)

Represents an attack projectile or object in the game. This could be a bubble, fin slap, or any other form of attack. Extends MovableObject to utilize movement capabilities and additional properties.

Constructor

new AttackObject(positionX, positionY, otherDirection, type, range)

Constructs an attack object with specified parameters, setting initial position, attack type, and range.
Parameters:
Name Type Description
positionX number The x-coordinate of the attack object's starting position.
positionY number The y-coordinate of the attack object's starting position.
otherDirection string The initial direction if not yet fired, typically left or right.
type string The type of attack, e.g., 'poison' or 'normal'.
range number The maximum distance the attack object can travel.
