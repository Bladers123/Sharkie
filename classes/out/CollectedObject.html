Class: CollectedObject

CollectedObject(imagePath, positionX, positionY, width, height, type)

Represents a collectible object in the game, such as a power-up or a coin. This class extends MovableObject.

Constructor

new CollectedObject(imagePath, positionX, positionY, width, height, type)

Constructs a new CollectedObject with specified image and dimensions.
Parameters:
Name Type Description
imagePath string The path to the image file for the object.
positionX number The x-coordinate of the object's initial position on the screen.
positionY number The y-coordinate of the object's initial position on the screen.
width number The width of the object.
height number The height of the object.
type string The type of the collectible object, used to differentiate between different kinds of collectibles.
