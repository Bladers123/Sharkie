Class: Level

Level(enemies, backgroundObjects, endBoss, collectedObjects, collectedAnimationObjects)

Represents a game level, containing all the elements that make up the gameplay environment, including enemies, the end boss, and various objects.

Constructor

new Level(enemies, backgroundObjects, endBoss, collectedObjects, collectedAnimationObjects)

Constructs a new Level instance with specified sets of game entities.
Parameters:
Name Type Description
enemies Array.<Enemy> An array of enemies present in the level.
backgroundObjects Array.<BackgroundObject> An array of background objects that make up the visual environment of the level.
endBoss Endboss The end boss of the level, typically a singular, significant enemy.
collectedObjects Array.<CollectedObject> An array of collectible objects that the player can interact with or collect.
collectedAnimationObjects Array.<CollectedAnimationObject> An array of animated collectible objects that add dynamic interactions.
